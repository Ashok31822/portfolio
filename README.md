## üì¶ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Ashok31822/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.

## üöÄ Deployment

To deploy this project to GitHub Pages:

1.  **Build and Deploy**
    ```bash
    npm run deploy
    ```
    This script will automatically build the project and push the `dist` folder to the `gh-pages` branch.
